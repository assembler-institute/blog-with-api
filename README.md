## Install

- Download repository
```javascript
  git clone git@github.com:alejandroaperez1994g/blog-with-api.git
  cd blog-with-api
```

- Install json-server
```javascript
  npm install
```

- Execute json-server

```javascript
  npm run server
```

## Starting

This is a project that simulates the display of posts brought from an API, being able to delete them and modify the content.
